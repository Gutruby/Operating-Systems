An operating systems controls the software and hardware of a computer. It is essential to operating a computer. An OS provides a graphical user interface with allows a user to create word documents and browse the interent as well as store media such as photos and music.
An operatiing systems creates a way to control hardware devices in the computer. Such as a plug and play device. If a device is not working properly the OS can help identify the issue and offer suggestions to resolve the issue, such as downloading drivers.
Another feature of an operating system is to allow the computer to access the internet. This can be done wired or wireless and the GUI is pretty easy to use. Alos the OS allows a way to connect bluetooth devices. I find my windows OS to be most useful to play games and access personal emails. As well, as get instant local news and sports information.
